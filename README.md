Myntra Clone
A fully responsive, front-end clone of the Myntra homepage. This project focuses on modern web layout techniques, including CSS Flexbox and responsive design principles.

 Live Demo
Click here to view the live project

 Tech Stack
HTML5: Semantic structure for e-commerce layouts.

CSS3: Flexbox for alignment, responsive media queries, and hover states.

GitHub Pages: Deployment and version control.

 Key Features
Fully Responsive: The layout adapts to different screen sizes (mobile, tablet, desktop) without breaking or overflow.

Modern Navigation: Interactive menu with hover effects.

Clean Architecture: Organized file structure and optimized image paths for performance.

Semantic SEO: Proper use of alt tags and structure for accessibility.

 Challenges & Learnings
Responsive Design: Implemented media queries to resolve layout issues on smaller screens.

File Management: Standardized directory naming conventions (category_images) to ensure cross-platform compatibility on Linux-based servers.

Git Workflow: Learned version control basics by managing and updating a repository.
